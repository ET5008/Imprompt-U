# Imprompt-U — Client

React 19 + TypeScript + Vite frontend for the Imprompt-U Socratic tutoring app.

See the [root README](../README.md) for full setup and project overview.

## Dev

```bash
npm install
npm run dev      # starts on http://localhost:5173, proxies /api to :3000
```

## Build

```bash
npm run build    # tsc + vite build → dist/
npm run preview  # preview production build locally
```

## Test

```bash
npm run test     # vitest
```
