Drop your Pimpy artwork files in this folder with these exact filenames:

  pimpy-happy.png       — default idle state (upload screen)
  pimpy-smiling.png     — waiting for user answer (questioning)
  pimpy-thinking.png    — AI processing / user typing / generating
  pimpy-sad.png         — user has been silent for 15+ seconds
  pimpy-crying.png      — (optional extra state)
  pimpy-delighted.png   — user finishes a good explanation / session complete

PNG format recommended. Any size works — they will be displayed at up to 180px.

Until these files are added, the app shows emoji placeholders automatically.
